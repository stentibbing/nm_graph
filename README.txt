== Description ==

This is simple plugin to implement a single graph for Nordic Milk website, using AmCharts4.

== Installation ==

1. Upload `nm_graph.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
